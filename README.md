# Digital-Alarm-Reminder-System
A Python console application for setting and managing time-based reminders. Features real-time clock checking using the datetime module, HH:MM format validation, and an automated alarm loop that triggers alerts when the set time is reached.
## Overview
A Python console application for setting and managing time-based reminders with real-time alarm checking.

## Features
- Display current date and time
- Add reminders with custom messages and time
- HH:MM time format validation
- Real-time alarm loop that checks every 60 seconds and triggers alerts

## How to Run
```bash
python digital_alarm.py
```

## Tools Used
- Python, datetime module, time module
